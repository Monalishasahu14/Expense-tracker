# Expense Tracker

A modern and responsive Expense Tracker app built with React JS to help you manage your daily expenses easily.
## Features
- Add, delete, and track your expenses
- Responsive design for all devices
- Simple and user-friendly interface
- View expense summary
## Tech Used
- React JS
- JavaScript, HTML, CSS
- GitHub Pages (for future live deployment)
